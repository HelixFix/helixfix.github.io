Application Desktop touristique

Une application JAVA qui recense des points d'intérêts d'une ville.
Desktop Development

# Programme & Code

  • ☑️ Identification de l'utlisateur (un user & un admin)
  • ☑️ Choix d'une ville et affichage de ses points d'intérêt
  • ☑️ Détails complets d'un point d'intérêt (dont photo)
  • ☑️ Ajout du point d'intérêt en favoris
  • ☑️ Retrait du point d'intérêt des favoris
  • ☑️ Affichage de la liste de favoris
  • ☑️ Ajout d'un point d'intérêt (dont import image)
  • ☑️ Contrôle des erreurs de saisie
  • ☑️ Qualité du code

# Fonctionnalités supplémentaires

  • Prise en compte de l'existant Filtre des points d'intérêt (au moins deux filtres fonctionnant simultanément)
  • ☑️ Retour en arrière possible
  • ☑️ Modification d'un point d'intérêt
  • ☑️ Un point d'intérêt possède une donnée sur son époque de construction un point d'intérêt possède une donnée sur sa catégorie (Château, Maison, Ruine, ...)
  • ☑️ Un point d'intérêt possède une donnée supplémentaire au choix (Charpente, Toiture, Fondations, Architecte, ...)
  • ☑️ Les points d'intérêts en favoris sont listés par ville
  • ☑️ Un point d'intérêt possède une photo principale et deux photos supplémentaires visibles dans les détails
  • ☑️ Gérer l'enregistrement des deux photos supplémentaires
  • ☑️ Utilisation d'une Base De Données relationnelle (MySql, MariaDB, Oracle SQL, ...)
  • ☑️ Un troisième utilisateur, l'Éditeur peux ajouter un point d'intérêt qui doit être validé par l'administrateur
  • Un nouvel utilisateur peut créer son compte qui sera activé par l'administrateur
